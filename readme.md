## 💻 Sobre o Projeto
Dashboard de Performance Bancária

Criar um painel de análise simulando indicadores de uma agência bancária ou carteira de clientes. Você pode usar dados fictícios ou públicos para analisar inadimplência, vendas, metas e perfil de clientes.

## 📚 Pré-requisitos de Habilidades e Níveis de Conhecimento

Antes de ingressar neste conteúdo, é necessário possuir conhecimento prévio nas seguintes áreas:

CORE SKILLS (essenciais):
Análise de dados
SQL
Visualização de dados e dashboards
NICE-TO-HAVE (complementares):
Estatística básica
Noções de Inteligência Artificial aplicada a dados

## 🛠️ Habilidades e Sub-habilidades que vamos aprender neste conteúdo

FERRAMENTAS E TECNOLOGIAS:
Excel
Power BI
SQL Server / PostgreSQL
Python
Google Sheets

## 🎯 Objetivos e Resultados Esperados

Após a conclusão do curso/projeto, os estudantes estarão aptos a:

ENTREGÁVEIS:
Dashboard no Power BI
Base de dados tratada em Excel ou SQL
Apresentação explicando insights encontrados

CRITÉRIOS DE ACEITAÇÃO:
Dashboard organizado e visualmente claro
Uso de KPIs relevantes
Explicação objetiva dos resultados encontrados
